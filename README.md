# ভেরিয়েবল (variable):

ভেরিয়েবল হল একটা ধারক বা পাত্রের মত।  কোন একটা জিনিস ভ্যারি করলে প্রোগামিংয়ের ভাষা ভেরিয়েবল বলে। Vary-ভ্যারি করে বা চেইঞ্জ হয়,
able-যোগ্য বা হতে পারে, variable- ভ্যারি করার যোগ্য বা চেইঞ্জ হতে পারে।
